# Audio Repository Structure

This folder contains the audio files for the Seva app.

## Structure
- **recitation/**: Contains the Sanskrit Shloka recitations.
- **explanation/**: Contains the Hindi meaning/explanation of the Shlokas.

## Naming Convention
Please follow this strict naming convention so we can easily automate the links later.

### Recitation (Sanskrit)
Format: `[scripture]_[chapter]_[verse].mp3`
Example: 
- `gita_1_1.mp3` (Bhagavad Gita, Chapter 1, Verse 1)
- `hanuman_1_1.mp3` (Hanuman Chalisa, Verse 1)

### Explanation (Hindi)
Format: `[scripture]_[chapter]_[verse]_explanation.mp3`
Example:
- `gita_1_1_explanation.mp3` (Hindi meaning of Gita Chapter 1, Verse 1)

## How to use
1. Record your audio.
2. Name the file according to the convention above.
3. Place it in the corresponding folder.
4. Push this folder to GitHub.
5. We will use the GitHub "Raw" links in the app's database.
